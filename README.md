### What's included:

- Angular 8+ & Typescript
- Bootstrap 4+ & SCSS
- RTL support
- High resolution

### How to install

-git clone https://github.com/anthonyerubio/Anthony-in-cali.git (or your repo)
-cd repo name
-npm install
-npm run start
-go to http://localhost:4200 in your local browser

### How to build
-npm run build
